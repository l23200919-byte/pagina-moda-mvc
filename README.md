# Página de Moda MVC - Macario Jiménez

Proyecto final de Programación Web.

## Tecnologías usadas
- HTML
- CSS
- Bootstrap
- JavaScript
- PHP
- MySQL
- XAMPP

## Funcionalidades
- Página principal de marca de moda
- Sección de empresa
- Colecciones
- Pasarela
- Shop
- Formulario de contacto
- CRUD de contactos con MySQL

## CRUD
- Crear contacto desde formulario
- Visualizar contactos en panel
- Editar contacto
- Eliminar contacto

## Base de datos
Base de datos: `macario_mvc`  
Tabla: `contactos`
